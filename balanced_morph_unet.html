Balanced Morphological U-Net

A standard U-Net whose two highest-resolution stages on each side are replaced by morphological neurons. Morphology handles the fine tubular geometry (vessels); the deep semantic stages stay convolutional. This is exactly the model --morph-unet balanced builds.

Depthwise-separable, on purpose: the 3×3 does the spatial morphology (one structuring element per channel); the 1×1 is the linear "activation" that mixes channels. The paper proves this linear step is what makes a morphological stack a universal approximator.

The MPM neuron

inside the depthwise morphology

out = α · ( δ + ε )
δ = bdil ∨ maxj( xj + se )  ← dilation
ε = bero ∧ minj( xj + se )  ← erosion
  • one shared 3×3 se for both paths — hard-tied, always
  • distinct biases bdil, bero — join / meet, not addition
  • max/min are soft via LogSumExp — β warms 2 → 10, dense gradients early
  • se initialised N(0,1) — paper's MPM init
